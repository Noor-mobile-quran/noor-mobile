## Design Context

### Users
**Universal Muslim audience** — all ages, global reach. Users open Noor as a daily spiritual companion, not a reference tool. Three behavioral archetypes:

- **Amina (Daily Reader):** 25-40, reads during commute and before bed. Needs frictionless beauty and accountability.
- **Yusuf (Listener):** 18-35, prefers audio recitation while driving or winding down. Needs immersive, audio-forward experience.
- **Fatima (Student):** 16-60, studies tafsir, bookmarks verses, takes notes. Needs structured reference tools.

**Job to be done:** Make consistent daily Quran engagement effortless and rewarding — turn occasional access into a daily habit.

**Emotional goals:** The interface should evoke calm focus, spiritual warmth, and gentle encouragement. Users should feel like they're opening a beautiful mushaf, not a tech product.

### Brand Personality
**Serene, Modern, Inviting**

- **Voice:** Warm companion, not cold interface. "Welcome back" not "Session resumed."
- **Tone:** Reverent toward sacred content, encouraging without guilt. Uses Islamic phrases naturally (Bismillah, Masha'Allah) where appropriate.
- **Never:** Gamify language around the Quran ("You unlocked a new surah!"), use guilt-based motivation, or assume a specific school of thought.

### Aesthetic Direction

**Visual tone:** Clean, warm minimalism with Islamic craftsmanship. Generous whitespace lets the Quran text breathe. Typography-first design.

**Color palette (established):**
- Cream: `#FFF9ED` (light bg), `#FFF3D6` (surface)
- Gold: `#D4A843` (primary accent), `#E8C547` (bright accent)
- Forest green: `#1B4332` (deep accent, Kaaba/Prophet's dome inspired)
- Sand: `#D4B896` (secondary text)
- Earth: `#6B5B45` (body text), `#2D261C` (headings)
- Dark mode: `#1A1410` (bg), `#2D261C` (surface) — warm midnight, never pure black

**Typography (established):**
- Arabic: Amiri serif — calligraphy-inspired, proper tashkeel rendering
- Latin: Inter — clean, modern, highly legible at all sizes

**Themes:** Light (cream/gold) + Dark (warm midnight) with CSS custom properties

**Visual identity:** Islamic geometric patterns, ornamental gold dividers, mosque silhouettes, crescent motifs, Kaaba-gradient hero, gold shimmer on special elements. Subtle and tasteful — never clip-art.

**Reference inspirations:**
- **Quran.com** — trusted, clean Quran reading layout
- **Headspace / Calm** — habit-forming daily engagement, beautiful onboarding, gentle encouragement
- **Apple Books / Kindle** — typography-first reading experience, distraction-free focus
- **Duolingo** — streak mechanics and daily return loops (adapt respectfully, not gamified)

**Anti-references:**
- No gamified / flashy aesthetics — no leaderboards, badges, neon colors, or achievement pop-ups. This is sacred content.

### Accessibility
- WCAG 2.1 AA compliance for all core experiences
- High contrast mode option for low-vision users
- Arabic Quran text must use proper `lang="ar"` and `dir="rtl"` attributes for screen reader announcement
- Touch targets minimum 44px on mobile
- Font size configurable (sm, md, lg, xl)

### Design Principles

1. **Sacred content, sacred space.** Every design decision respects that the Quran is not ordinary content. No gamification language, no flashy animations competing with the text. The UI serves the words, not the other way around.

2. **Typography is the interface.** Arabic calligraphy and clean Latin type are the primary design elements. Get the reading experience right and everything else follows. Generous whitespace, proper RTL, correct tashkeel.

3. **Habit through beauty, not pressure.** Engagement mechanics (streaks, goals, daily ayah) motivate through gentle encouragement and visual reward — never guilt, urgency, or competition. "You read 3 ayahs today" not "You only read 3 ayahs."

4. **Warm minimalism.** Clean and uncluttered, but never cold or sterile. The cream/gold/earth palette and Islamic ornamental touches give the interface soul. Every decorative element is intentional.

5. **Adapt to the reader.** Three UX modes (Serene, Immersive, Study), configurable font sizes, light/dark themes, and high contrast support. The app meets each user where they are without requiring them to compromise.
